## Usage
Compile the code with
```
lein uberjar
```

and then run it with
```
java -jar target/uberjar/terra-tracker-api-0.1.0-SNAPSHOT-standalone.jar
```

The Swagger API doc page can then be accessed at
```
http://localhost:3001/api
```
